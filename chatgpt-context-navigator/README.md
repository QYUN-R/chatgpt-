# AI Context Navigator (ChatGPT / Gemini / Grok)

这是一个简单的浏览器扩展，用于在 ChatGPT、Google Gemini 和 Grok (x.com) 页面右侧生成一个问题目录，点击即可跳转到对应的对话位置。

## 支持平台
- **ChatGPT** (chatgpt.com / chat.openai.com)
- **Google Gemini** (gemini.google.com)
- **Grok** (x.com / grok.com)

## 安装方法 (Chrome / Edge / Brave)

1. **打开扩展程序管理页面**：
   - Chrome/Brave: 在地址栏输入 `chrome://extensions/`
   - Edge: 在地址栏输入 `edge://extensions/`

2. **开启开发者模式**：
   - 在页面右上角（或左下角）找到 "开发者模式" (Developer mode) 开关并打开。

3. **加载已解压的扩展程序**：
   - 点击出现的 "加载已解压的扩展程序" (Load unpacked) 按钮。
   - 选择本文件夹 `chatgpt-context-navigator`。

4. **完成！**
   - 刷新 ChatGPT、Gemini 或 Grok 页面，你应该能看到右侧出现了一个悬浮的目录框。

## 功能
- 自动提取所有的 User Prompt (我的提问)。
- 点击目录项平滑滚动定位。
- 支持深色/浅色模式。
- 自动识别当前使用的 AI 平台。

## 注意事项
由于 Gemini 和 Grok 的网页结构经常更新，如果导航栏无法显示内容，可能需要更新选择器配置。
